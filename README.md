Refaccionaria Demo — Sitio Web Responsivo

Proyecto web profesional para una refaccionaria de motos, diseñado como demo real para negocio local y como parte de portafolio web.
Incluye navegación adaptable, catálogo dinámico con datos en JSON, modo oscuro, componentes modulares y optimización para móviles.

Vista previa del proyecto

Demo en línea: https://refaccionaria-demo.netlify.app/

Características principales

Diseño responsivo para celulares, tablets y computadoras.

Modo oscuro con guardado automático en localStorage.

Catálogo dinámico cargado desde productos.json.

Buscador de refacciones en tiempo real.

Componentes reutilizables (header.html y footer.html cargados con fetch).

Optimización de rendimiento:

Carga diferida (loading="lazy") de imágenes

Código modular

Estructura limpia para ampliar el proyecto

Botón directo a WhatsApp para cotizaciones.

Tecnologías utilizadas

HTML5

CSS3

JavaScript (Vanilla)

JSON para datos del catálogo

Git y GitHub para control de versiones

Netlify para despliegue

Cómo ejecutar el proyecto localmente

Clona el repositorio

git clone https://github.com/Cristopher-Delao/refaccionaria-demo.git


Entra a la carpeta

cd refaccionaria-demo


Inicia un servidor local (Visual Studio Code):

Instala la extensión Live Server

Click derecho → Open with Live Server
